دلیل انتخاب موضوع:علاقه به کتاب و کتاب فروشی و تمایل به ساخت سیستمی حول محور مدیریت کتاب


هدف: ساخت برنامه ای برای مدیریت موجودی کتاب های یک فروشگاه و مرتب سازی کتاب ها بر اساس ویژگی هاشون و مدیریت سیستم امانت و بازگشت کتاب


ویژگی ها:

1-اضافه کردن کتاب
2-گرفتن خصوصیات کتاب مثل (اسم کتاب/نام نویسنده/ژانر/کشور مبدا) از کاربر و مشخص کردن اینکه آیا کتاب برای امانت هست یا نه
3-پروژه منو محور
4-ذخیره سازی اطلاعات در فایل تکست
5-امکان جستجو کتاب بر اساس اسم
6-گرفتن نام کاربر هنگام ورود
7-ذخیره نام و شماره شخصی که امانت گرفته
